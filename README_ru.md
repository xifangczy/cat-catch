# 📑 Описание

**Кошачья Лапа (cat-catch)** — это расширение для перехвата и анализа ресурсов. Оно помогает вам находить и скачивать все ресурсы (видео, m3u8, изображения и т.д.) на текущей странице.

# 📘 Способы установки

## Установка из магазина расширений
Просто перейдите по ссылкам выше в официальный магазин и установите расширение.

## Установка из исходного кода
1. Склонируйте репозиторий (`git clone`).
2. Откройте страницу управления расширениями и включите **«Режим разработчика»**.
3. Нажмите **«Загрузить распакованное расширение»** и выберите папку с расширением.

## Установка через .crx файл
1. Перейдите в раздел [Releases](https://github.com/xifangczy/cat-catch/releases), правой кнопкой мыши нажмите на crx-файл и выберите **«Сохранить как»**.
2. Откройте страницу расширений, включите **«Режим разработчика»**.
3. Перетащите скачанный .crx файл в окно расширений.

# 📚 Совместимость
Начиная с версии 1.0.17 требуется Chromium-ядро версии **93 и выше**.  
Для версий ниже 93 используйте 1.0.16.  
Для полного набора функций рекомендуется Chromium 104 и новее.
